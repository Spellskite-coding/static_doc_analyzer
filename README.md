Bonjour ! Ce petit script Python est un outil de triage rapide qui fouille vos documents suspects (Word, Excel, PDF) pour y débusquer d'éventuelles menaces cachées, le tout sans jamais les exécuter. Pour lancer l'analyse tapez simplement : `python3 static_doc_analyzer.py <chemin_du_document>`.

Le script vous dira tout de suite si votre fichier est sain où malveillant !
